1.3 Web Proxyサーバの登録
======================================

BIG-IPのGUIにログインします。

1. 「Local Traffic」 → 「Pools」で表示された画面の右上にある\ ``Create``\ ボタンをクリックします。


.. figure:: images/Picture1.png
   :scale: 50%
   :align: center



2. 赤枠の必要内容を入力し、\ ``Finished``\ ボタンをクリックします。

.. figure:: images/Picture2.png
   :scale: 50%
   :align: center
